## ajs-boom:
Easily create AngularJS modules and components

## Example:

```bash
$ node create-module --name=loja-livros --components=lista-livros,livro,carrinho,categorias --path=/app/
```

## Options:
|               | Type          | Optional |
| ------------- |:-------------:| --------:|
| name          | string        | no       |
| components    | string        | yes      |
| path          | string        | yes      |
